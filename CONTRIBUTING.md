# Contributing to Feishu Webhook Skill

Thank you for your interest in contributing to this project!

## How to Contribute

### Reporting Issues

- Use GitHub Issues to report bugs or suggest features
- Provide clear descriptions and steps to reproduce issues
- Include relevant environment information

### Pull Requests

1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes
4. Test your changes thoroughly
5. Submit a pull request

### Code Style

- Follow existing code patterns
- Write clear, descriptive commit messages
- Document new features or changes

### Documentation

- Update relevant `.md` files when adding features
- Keep examples up to date
- Write in both English and Chinese where appropriate

## Development Setup

```bash
# Clone your fork
git clone https://github.com/YOUR_USERNAME/feishu-webhook-skill.git
cd feishu-webhook-skill

# Install Python dependencies (for upload script)
pip install requests requests-toolbelt
```
